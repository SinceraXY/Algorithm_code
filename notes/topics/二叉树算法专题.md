## 🎯 二叉树基础

### 二叉树特性
- **层次结构**: 树形结构，每个节点最多有两个子节点
- **递归定义**: 二叉树是递归结构，左右子树也是二叉树
- **非线性**: 不同于数组和链表的线性结构
- **层序关系**: 节点之间有明确的父子、兄弟关系

### 二叉树类型
| 类型 | 特点 | 应用场景 |
|------|------|----------|
| **满二叉树** | 每层节点数达到最大值 | 完美平衡 |
| **完全二叉树** | 除最后一层外都是满的 | 堆的实现 |
| **二叉搜索树** | 左<根<右的有序性 | 查找、排序 |
| **平衡二叉树** | 左右子树高度差≤1 | AVL树、红黑树 |
| **线索二叉树** | 利用空指针存储遍历信息 | 遍历优化 |

### 满二叉树与完全二叉树（重点）
- **满二叉树**：如果一棵二叉树只有度为 0 的结点和度为 2 的结点，并且度为 0 的结点在同一层上，则这棵二叉树为满二叉树。
  - 常见等价描述：深度为 `k` 的满二叉树，一共有 `2^k - 1` 个节点。
- **完全二叉树**：在完全二叉树中，除了最底层节点可能没填满外，其余每层节点数都达到最大值，并且最下面一层的节点都集中在该层最左边的若干位置。
  - ⚠️ 很多人容易“看起来像”就判断完全二叉树，建议在脑中严格对齐定义：最后一层必须从左到右连续填充，中间不能出现空缺。
  - 若最底层为第 `h` 层，则该层包含 `1 ~ 2^h` 个节点。

### 堆与完全二叉树
- **堆（heap）**本质是一棵**完全二叉树**，并满足父子节点的顺序关系：
  - 父节点 ≥ 子节点：大顶堆
  - 父节点 ≤ 子节点：小顶堆
- 因为堆是完全二叉树，所以非常适合用数组表示（便于用下标计算父子关系）。

### 二叉搜索树（BST）与平衡二叉搜索树
- **BST（有序树）**：
  - 若左子树不空，则左子树上所有结点的值均小于根结点的值
  - 若右子树不空，则右子树上所有结点的值均大于根结点的值
  - 左、右子树也分别为二叉搜索树
- **平衡二叉搜索树（AVL 一类定义）**：左右两个子树的高度差的绝对值不超过 1，并且左右两个子树都是一棵平衡二叉树。

**容器底层与复杂度（C++）**：
- `map`、`set`、`multimap`、`multiset` 的底层通常是平衡二叉搜索树（如红黑树），因此增删查为 `O(log n)`。
- `unordered_map`、`unordered_set` 的底层是哈希表，平均增删查为 `O(1)`。

### 二叉树的存储方式
- **链式存储**：用指针连接左右孩子（刷题/面试最常见的表示）。
- **顺序存储（数组）**：用数组表示二叉树（常用于完全二叉树/堆）。
  - 若父节点下标为 `i`：
    - 左孩子下标：`2 * i + 1`
    - 右孩子下标：`2 * i + 2`

在 LeetCode 中，二叉树节点定义通常已给出；但现场面试可能需要手写 `TreeNode`，建议把这种基础结构做到“白纸也能写出来”。

| 操作 | 平均时间 | 最坏时间 | 说明 |
|------|---------|---------|------|
| 搜索 | O(log n) | O(n) | BST平衡时O(log n) |

| 删除 | O(log n) | O(n) | 需要找到节点并调整 |

## 🔄 二叉树遍历

### 核心思想
二叉树遍历是访问树中所有节点的过程，根据访问根节点的顺序不同，分为前序、中序、后序遍历。

### DFS 与 BFS 框架
- **深度优先遍历（DFS）**：先往深走，遇到叶子节点再往回走。
  - 常见落地：前序 / 中序 / 后序遍历。
  - 递归写法本质上借助了“栈”（系统调用栈）；也可以改写成显式栈的迭代写法。
- **广度优先遍历（BFS）**：一层一层遍历。
  - 常见落地：层序遍历。
  - 迭代实现通常使用队列（先进先出）。

**前中后序口诀**：前/中/后序指的是“中间节点（根节点）”的位置。
- 前序遍历：中左右
- 中序遍历：左中右
- 后序遍历：左右中

### 遍历方式对比
| 遍历方式 | 访问顺序 | 应用场景 | 特点 |
|---------|---------|---------|------|
| **前序遍历** | 根→左→右 | 复制树、前缀表达式 | 先处理根节点 |
| **中序遍历** | 左→根→右 | BST排序、中缀表达式 | BST得到有序序列 |
| **后序遍历** | 左→右→根 | 删除树、后缀表达式 | 先处理子节点 |
| **层序遍历** | 逐层访问 | 最短路径、层次结构 | 使用队列实现 |

### 递归写法的方法论（递归三要素）
本篇将介绍前/中/后序的递归写法。一些同学可能会感觉很简单，其实不然——我们要通过简单题目把方法论确定下来，有了方法论，后面才能应付复杂的递归。

这里帮助大家确定下来递归算法的三个要素：**每次写递归，都按照这三要素来写，可以保证写出正确的递归算法**。

- **确定递归函数的参数和返回值**：确定哪些参数是递归过程中需要处理的，那么就在递归函数里加上这个参数；并且要明确每次递归的返回值是什么，从而确定递归函数的返回类型。
- **确定终止条件**：写完递归算法运行时，经常会遇到栈溢出错误，就是没写终止条件或者终止条件写的不对。操作系统用栈来保存每一层递归信息，如果递归没有终止，内存栈必然会溢出。
- **确定单层递归的逻辑**：确定每一层递归需要处理的信息，也就是在这一层做什么；在这里会重复调用自己来实现递归过程。

### 以前序遍历为例（把三要素落到代码上）
- **确定递归函数的参数和返回值**：因为要收集前序遍历节点数值，所以参数里需要传入 `vector<int>&` 来存节点数值；除此之外就不需要处理其他数据，也不需要返回值，因此递归函数返回类型为 `void`。

  ```cpp
  void traversal(TreeNode* cur, vector<int>& vec)
  ```

- **确定终止条件**：当前遍历节点为空，本层递归结束。

  ```cpp
  if (cur == nullptr) return;
  ```

- **确定单层递归逻辑**：前序遍历顺序是“中左右”，因此本层逻辑如下。

  ```cpp
  vec.push_back(cur->val);    // 中
  traversal(cur->left, vec);  // 左
  traversal(cur->right, vec); // 右
  ```

### 迭代遍历模板
 
```cpp
// 前序遍历 - 迭代法
vector<int> preorderTraversal(TreeNode* root) {
    vector<int> result;
    if (root == nullptr) return result;
    
    stack<TreeNode*> st;
    st.push(root);
    
    while (!st.empty()) {
        TreeNode* node = st.top();
        st.pop();
        result.push_back(node->val);
        
        // 先压右子树，再压左子树（栈是后进先出）
        if (node->right) st.push(node->right);
        if (node->left) st.push(node->left);
    }
    
    return result;
}

// 中序遍历 - 迭代法
vector<int> inorderTraversal(TreeNode* root) {
    vector<int> result;
    stack<TreeNode*> st;
    TreeNode* cur = root;
    
    while (cur != nullptr || !st.empty()) {
        // 一直向左走到底
        while (cur != nullptr) {
            st.push(cur);
            cur = cur->left;
        }
        
        // 处理栈顶节点
        cur = st.top();
        st.pop();
        result.push_back(cur->val);
        
        // 转向右子树
        cur = cur->right;
    }
    
    return result;
}

// 后序遍历 - 迭代法
vector<int> postorderTraversal(TreeNode* root) {
    vector<int> result;
    if (root == nullptr) return result;
    
    stack<TreeNode*> st;
    st.push(root);
    
    while (!st.empty()) {
        TreeNode* node = st.top();
        st.pop();
        result.push_back(node->val);
        
        // 先压左子树，再压右子树
        if (node->left) st.push(node->left);
        if (node->right) st.push(node->right);
    }
    
    // 反转结果（根→右→左 变为 左→右→根）
    reverse(result.begin(), result.end());
    return result;
}
```

### 层序遍历

```cpp
// LeetCode 102: 二叉树的层序遍历
vector<vector<int>> levelOrder(TreeNode* root) {
    vector<vector<int>> result;
    if (root == nullptr) return result;
    
    queue<TreeNode*> q;
    q.push(root);
    
    while (!q.empty()) {
        int size = q.size();
        vector<int> level;
        
        // 处理当前层的所有节点
        for (int i = 0; i < size; i++) {
            TreeNode* node = q.front();
            q.pop();
            level.push_back(node->val);
            
            if (node->left) q.push(node->left);
            if (node->right) q.push(node->right);
        }
        
        result.push_back(level);
    }
    
    return result;
}
```

## 🏗️ 二叉树构造

### 核心思想
通过遍历序列重建二叉树，关键是找到根节点位置，然后递归构造左右子树。

### 构造方法对比
| 遍历组合 | 能否唯一确定 | 关键点 |
|---------|-------------|--------|
| 前序+中序 | ✅ 能 | 前序确定根，中序划分左右 |
| 后序+中序 | ✅ 能 | 后序确定根，中序划分左右 |
| 前序+后序 | ❌ 不能 | 无法确定左右子树边界 |
| 层序+中序 | ✅ 能 | 层序确定根，中序划分左右 |


### 从前序和中序构造

```cpp
// LeetCode 105: 从前序与中序遍历序列构造二叉树
class Solution {
public:
    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        // 建立中序遍历的值到索引的映射
        unordered_map<int, int> inMap;
        for (int i = 0; i < inorder.size(); i++) {
            inMap[inorder[i]] = i;
        }
        return build(preorder, 0, preorder.size() - 1,
                    inorder, 0, inorder.size() - 1, inMap);
    }
    
private:
    TreeNode* build(vector<int>& preorder, int preStart, int preEnd,
                   vector<int>& inorder, int inStart, int inEnd,
                   unordered_map<int, int>& inMap) {
        if (preStart > preEnd) return nullptr;
        
        // 前序遍历的第一个元素是根节点
        int rootVal = preorder[preStart];
        TreeNode* root = new TreeNode(rootVal);
        
        // 在中序遍历中找到根节点的位置
        int inRoot = inMap[rootVal];
        int leftSize = inRoot - inStart;
        
        // 递归构造左右子树
        root->left = build(preorder, preStart + 1, preStart + leftSize,
                          inorder, inStart, inRoot - 1, inMap);
        root->right = build(preorder, preStart + leftSize + 1, preEnd,
                           inorder, inRoot + 1, inEnd, inMap);
        
        return root;
    }
};
```

### 从中序和后序构造

```cpp
// LeetCode 106: 从中序与后序遍历序列构造二叉树
TreeNode* buildTree(vector<int>& inorder, vector<int>& postorder) {
    unordered_map<int, int> inMap;
    for (int i = 0; i < inorder.size(); i++) {
        inMap[inorder[i]] = i;
    }
    return build(inorder, 0, inorder.size() - 1,
                postorder, 0, postorder.size() - 1, inMap);
}

TreeNode* build(vector<int>& inorder, int inStart, int inEnd,
               vector<int>& postorder, int postStart, int postEnd,
               unordered_map<int, int>& inMap) {
    if (inStart > inEnd) return nullptr;
    
    // 后序遍历的最后一个元素是根节点
    int rootVal = postorder[postEnd];
    TreeNode* root = new TreeNode(rootVal);
    
    int inRoot = inMap[rootVal];
    int leftSize = inRoot - inStart;
    
    root->left = build(inorder, inStart, inRoot - 1,
                      postorder, postStart, postStart + leftSize - 1, inMap);
    root->right = build(inorder, inRoot + 1, inEnd,
                       postorder, postStart + leftSize, postEnd - 1, inMap);
    
    return root;
}
```

```cpp
// 二叉树的最大深度 (LeetCode 104)
int maxDepth(TreeNode* root) {
    if (root == nullptr) return 0;
    return max(maxDepth(root->left), maxDepth(root->right)) + 1;
}

// 二叉树的最小深度 (LeetCode 111)
int minDepth(TreeNode* root) {
    if (root == nullptr) return 0;
    if (root->left == nullptr) return minDepth(root->right) + 1;
    if (root->right == nullptr) return minDepth(root->left) + 1;
    return min(minDepth(root->left), minDepth(root->right)) + 1;
}
```


### 平衡性判断

```cpp
// 平衡二叉树 (LeetCode 110)
class Solution {
public:
    bool isBalanced(TreeNode* root) {
        return getHeight(root) != -1;
    }
    
private:
    int getHeight(TreeNode* node) {
        if (node == nullptr) return 0;
        
        int leftHeight = getHeight(node->left);
        if (leftHeight == -1) return -1;  // 左子树不平衡
        
        int rightHeight = getHeight(node->right);
        if (rightHeight == -1) return -1;  // 右子树不平衡
        
        // 检查当前节点是否平衡
        if (abs(leftHeight - rightHeight) > 1) return -1;
        
        return max(leftHeight, rightHeight) + 1;
    }
};
```

### 节点数量

```cpp
// 完全二叉树的节点个数 (LeetCode 222)
int countNodes(TreeNode* root) {
    if (root == nullptr) return 0;
    
    int leftDepth = 0, rightDepth = 0;
    TreeNode* left = root->left;
    TreeNode* right = root->right;
    
    // 计算左右子树深度
    while (left) {
        left = left->left;
        leftDepth++;
    }
    while (right) {
        right = right->right;
        rightDepth++;
    }
    
    // 如果左右深度相同，说明是满二叉树
    if (leftDepth == rightDepth) {
        return (2 << leftDepth) - 1;  // 2^(depth+1) - 1
    }
    
    // 否则递归计算
    return countNodes(root->left) + countNodes(root->right) + 1;
}
```

## 🔍 二叉搜索树

### BST特性
- **有序性**: 左子树所有节点 < 根节点 < 右子树所有节点
- **中序遍历**: 得到递增序列
- **查找效率**: 平衡时O(log n)，退化时O(n)

### BST验证

```cpp
// 验证二叉搜索树 (LeetCode 98)
class Solution {
public:
    bool isValidBST(TreeNode* root) {
        return validate(root, LONG_MIN, LONG_MAX);
    }
    
private:
    bool validate(TreeNode* node, long minVal, long maxVal) {
        if (node == nullptr) return true;
        
        // 当前节点值必须在(minVal, maxVal)范围内
        if (node->val <= minVal || node->val >= maxVal) {
            return false;
        }
        
        // 递归验证左右子树
        return validate(node->left, minVal, node->val) &&
               validate(node->right, node->val, maxVal);
    }
};
```

### BST查找

```cpp
// 二叉搜索树中的搜索 (LeetCode 700)
TreeNode* searchBST(TreeNode* root, int val) {
    if (root == nullptr || root->val == val) return root;
    
    // 利用BST的有序性
    if (val < root->val) {
        return searchBST(root->left, val);
    } else {
        return searchBST(root->right, val);
    }
}
```

```cpp
TreeNode* insertIntoBST(TreeNode* root, int val) {
```

### BST删除

```cpp
// 删除二叉搜索树中的节点 (LeetCode 450)
TreeNode* deleteNode(TreeNode* root, int key) {
    if (root == nullptr) return nullptr;
    
    if (key < root->val) {
        root->left = deleteNode(root->left, key);
    } else if (key > root->val) {
        root->right = deleteNode(root->right, key);
    } else {
        // 找到要删除的节点
        if (root->left == nullptr) return root->right;
        if (root->right == nullptr) return root->left;
        
        // 节点有两个子节点：找右子树的最小节点替换
        TreeNode* minNode = getMin(root->right);
        root->val = minNode->val;
        root->right = deleteNode(root->right, minNode->val);
    }
    
    return root;
}

TreeNode* getMin(TreeNode* node) {
    while (node->left != nullptr) {
        node = node->left;
    }
    return node;
}
```

### BST转换

```cpp
// 将有序数组转换为二叉搜索树 (LeetCode 108)
TreeNode* sortedArrayToBST(vector<int>& nums) {
    return build(nums, 0, nums.size() - 1);
}

TreeNode* build(vector<int>& nums, int left, int right) {
    if (left > right) return nullptr;
    
    // 选择中间元素作为根节点，保证平衡
    int mid = left + (right - left) / 2;
    TreeNode* root = new TreeNode(nums[mid]);
    
    root->left = build(nums, left, mid - 1);
    root->right = build(nums, mid + 1, right);
    
    return root;
}
```

### 路径总和

```cpp
// 路径总和 (LeetCode 112)
bool hasPathSum(TreeNode* root, int targetSum) {
    if (root == nullptr) return false;
    
    // 叶子节点
    if (root->left == nullptr && root->right == nullptr) {
        return root->val == targetSum;
    }
    
    // 递归检查左右子树
    return hasPathSum(root->left, targetSum - root->val) ||
           hasPathSum(root->right, targetSum - root->val);
}
```

### 路径总和 II

```cpp
// 路径总和 II (LeetCode 113) - 找出所有路径
class Solution {
public:
    vector<vector<int>> pathSum(TreeNode* root, int targetSum) {
        vector<vector<int>> result;
        vector<int> path;
        dfs(root, targetSum, path, result);
        return result;
    }
    
private:
    void dfs(TreeNode* node, int sum, vector<int>& path, 
             vector<vector<int>>& result) {
        if (node == nullptr) return;
        
        path.push_back(node->val);
        
        // 叶子节点且路径和等于目标值
        if (node->left == nullptr && node->right == nullptr && 
            sum == node->val) {
            result.push_back(path);
        }
        
        dfs(node->left, sum - node->val, path, result);
        dfs(node->right, sum - node->val, path, result);
        
        path.pop_back();  // 回溯
    }
};
```


### 二叉树的最大路径和

```cpp
// 二叉树中的最大路径和 (LeetCode 124)
class Solution {
public:
    int maxPathSum(TreeNode* root) {
        int maxSum = INT_MIN;
        maxGain(root, maxSum);
        return maxSum;
    }
    
private:
    int maxGain(TreeNode* node, int& maxSum) {
        if (node == nullptr) return 0;
        
        // 递归计算左右子树的最大贡献值（负数不选）
        int leftGain = max(maxGain(node->left, maxSum), 0);
        int rightGain = max(maxGain(node->right, maxSum), 0);
        
        // 更新最大路径和（经过当前节点的路径）
        int pathSum = node->val + leftGain + rightGain;
        maxSum = max(maxSum, pathSum);
        
        // 返回当前节点的最大贡献值
        return node->val + max(leftGain, rightGain);
    }
};
```

## 👨‍👩‍👦 公共祖先

### 最近公共祖先

```cpp
// 二叉树的最近公共祖先 (LeetCode 236)
TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
    // 递归终止条件
    if (root == nullptr || root == p || root == q) {
        return root;
    }
    
    // 递归查找左右子树
    TreeNode* left = lowestCommonAncestor(root->left, p, q);
    TreeNode* right = lowestCommonAncestor(root->right, p, q);
    
    // 如果p和q分别在左右子树，当前节点就是LCA
    if (left != nullptr && right != nullptr) {
        return root;
    }
    
    // 否则返回非空的那个子树结果
    return left != nullptr ? left : right;
}
```

### BST的最近公共祖先

```cpp
// 二叉搜索树的最近公共祖先 (LeetCode 235)
TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
    // 利用BST的性质
    if (root->val > p->val && root->val > q->val) {
        // p和q都在左子树
        return lowestCommonAncestor(root->left, p, q);
    }
    if (root->val < p->val && root->val < q->val) {
        // p和q都在右子树
        return lowestCommonAncestor(root->right, p, q);
    }
    // p和q分别在左右子树，或其中一个就是root
    return root;
}
```

```cpp
TreeNode* invertTree(TreeNode* root) {
    if (root == nullptr) return nullptr;
    
    // 交换左右子树
```

```cpp
// 对称二叉树 (LeetCode 101)
class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        if (root == nullptr) return true;
        return isMirror(root->left, root->right);
    }
    
private:
    bool isMirror(TreeNode* left, TreeNode* right) {
        if (left == nullptr && right == nullptr) return true;
        if (left == nullptr || right == nullptr) return false;
        
        return (left->val == right->val) &&
               isMirror(left->left, right->right) &&
               isMirror(left->right, right->left);
    }
};
```

```cpp
// 合并二叉树 (LeetCode 617)
TreeNode* mergeTrees(TreeNode* root1, TreeNode* root2) {
    if (root1 == nullptr) return root2;
    if (root2 == nullptr) return root1;
    
    // 合并当前节点
    root1->val += root2->val;
    
    // 递归合并左右子树
    root1->left = mergeTrees(root1->left, root2->left);
    root1->right = mergeTrees(root1->right, root2->right);
    
    return root1;
}
```

## 🔧 解题技巧总结

### 1. 题目类型识别
- **遍历问题** → 递归/迭代/层序遍历
- **构造问题** → 递归+哈希表优化
- **属性判断** → 递归计算深度/平衡性
- **BST问题** → 利用有序性质
- **路径问题** → DFS+回溯
- **祖先问题** → 递归查找

### 2. 递归三要素
1. **确定递归函数参数和返回值**
   - 需要什么参数？返回什么类型？
2. **确定终止条件**
   - 什么时候停止递归？
3. **确定单层递归逻辑**
   - 当前层做什么？如何调用下一层？

### 3. 常用技巧
- **前序位置**: 在进入节点时处理（自顶向下）
- **后序位置**: 在离开节点时处理（自底向上）
- **中序位置**: BST中获取有序序列
- **层序遍历**: 使用队列，适合层次相关问题
- **哈希表**: 优化查找，如构造树时查找根节点位置

### 4. 时间复杂度分析
- **遍历**: O(n) - 访问每个节点一次
- **查找**: O(log n) ~ O(n) - 取决于树的平衡性
- **构造**: O(n) - 需要处理所有节点
- **路径**: O(n) - 可能需要遍历所有路径

### 5. 空间复杂度分析
- **递归栈**: O(h) - h为树的高度
- **迭代栈**: O(h) - 最坏情况下存储一条路径
- **队列**: O(w) - w为树的最大宽度
- **哈希表**: O(n) - 存储节点值到索引的映射


### 7. 递归与迭代的深度对比

在算法竞赛和实际项目开发中，选择递归还是迭代不仅是代码风格的问题，更涉及性能与稳定性的权衡：

- **时间复杂度**：在不考虑函数调用开销的前提下，两者基本一致。但递归涉及频繁的**函数调用开销**（参数压栈、返回地址保存等），在海量数据下可能略慢。
- **空间复杂度**：
    - **递归法**：由于需要系统堆栈存储每一层的参数、局部变量和返回值，空间开销较大，且受限于系统栈大小。
    - **迭代法**：通常显式维护一个栈或队列，虽然也消耗空间，但其分配在堆上（或受手动控制），容量远大于系统栈。
- **稳定性与收敛性**：
    - 递归虽然代码简洁、更符合人类逻辑，但**收敛性较差**。若递归深度过大且没有有效的剪枝，极易导致**栈溢出（Stack Overflow）**。
- **项目实践建议**：
    - 「在实际项目开发中，应尽量避免不加控制的递归！」
    - 项目代码的参数和调用关系往往非常复杂，递归深度难以精确预估。在处理大规模数据或深度未知的树结构时，优先考虑**迭代法**或**尾递归优化**（如果语言支持）。
    - 总结一句话：**递归方便了程序员（思维负担轻），但难为了机器（资源消耗多且风险高）。**

## 💡 实战技巧

### 递归 vs 迭代选择
```
递归优势：
✅ 代码简洁，易于理解
✅ 符合树的递归定义
✅ 适合大多数树的问题

迭代优势：
✅ 避免栈溢出（深度很大时）
✅ 空间可控
✅ 某些场景更高效
```

```cpp
void printTree(TreeNode* root) {
    if (root == nullptr) {
        cout << "Empty tree" << endl;
        return;
    }
    
    queue<TreeNode*> q;
    q.push(root);
    
    while (!q.empty()) {
        int size = q.size();
        for (int i = 0; i < size; i++) {

            q.pop();
            
            if (node) {
                cout << node->val << " ";
```

### 常见陷阱
1. **空指针**: 访问节点前检查是否为空

4. **迭代代替递归**: 深度很大时避免栈溢出

## 🌟 核心思想总结

### 递归思维
二叉树天然具有递归结构，大部分问题都可以用递归解决：
- **分解问题**: 将问题分解为左右子树的子问题
- **合并结果**: 根据子问题的结果得到原问题的解
- **终止条件**: 空节点或叶子节点

### 遍历框架
```cpp
void traverse(TreeNode* root) {
    if (root == nullptr) return;
    
    // 前序位置：进入节点时的操作
    // 适合自顶向下传递信息
    
    traverse(root->left);
    
    // 中序位置：在左右子树之间的操作
    // BST中序遍历得到有序序列
    
    traverse(root->right);
    
    // 后序位置：离开节点时的操作
    // 适合自底向上收集信息
}
```

### 分治思想
很多树的问题可以用分治法解决：
1. **分解**: 将问题分解为左右子树的子问题
2. **解决**: 递归解决子问题
3. **合并**: 合并子问题的解得到原问题的解
