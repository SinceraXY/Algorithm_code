# 222. 完全二叉树的节点个数

- **平台**：LeetCode
- **链接**：https://leetcode.cn/problems/count-complete-tree-nodes/description/
- **难度**：Easy
- **标签**：位运算、树、二分查找、二叉树

## 题目描述

给你一棵完全二叉树的根节点 `root`，求出该树的节点个数。

完全二叉树的定义如下：除了最底层节点可能没有填满外，其余每层节点数都达到最大值，并且最底层的节点都集中在该层最左边的若干位置。

## 示例

**示例 1：**

```text
输入：root = [1,2,3,4,5,6]
输出：6
```

**示例 2：**

```text
输入：root = []
输出：0
```

**示例 3：**

```text
输入：root = [1]
输出：1
```

## 解题思路

当前文件夹内的代码实现了两种通用二叉树计数方法：

- 递归统计节点数量；
- 层序遍历统计节点数量。

这两种方法不依赖完全二叉树的特殊性质，因此对普通二叉树也同样适用。

### 方法一：递归

#### 思路与算法

定义 `getNodesNum(cur)` 表示以 `cur` 为根节点的子树节点个数。

递归终止条件：

```text
如果 cur 为空，返回 0
```

递归统计左右子树节点数量：

```text
leftNum = getNodesNum(cur.left)
rightNum = getNodesNum(cur.right)
```

当前子树节点总数为：

```text
leftNum + rightNum + 1
```

其中 `+1` 表示当前节点本身。

### 方法二：迭代法（层序遍历）

#### 思路与算法

使用队列进行广度优先搜索。

初始时，如果根节点不为空，就将根节点入队。之后不断从队列中取出节点：

1. 每取出一个节点，计数 `result` 加 `1`；
2. 如果该节点有左孩子，将左孩子入队；
3. 如果该节点有右孩子，将右孩子入队；
4. 队列为空时，所有节点都已经统计完成。

最终 `result` 就是整棵树的节点个数。

## 复杂度分析

设二叉树节点数为 `n`，树高为 `h`。

### 递归

- **时间复杂度**：`O(n)`，每个节点都会被访问一次。
- **空间复杂度**：`O(h)`，主要来自递归调用栈。平衡情况下为 `O(log n)`，最坏情况下树退化成链状，为 `O(n)`。

### 迭代法（层序遍历）

- **时间复杂度**：`O(n)`，每个节点都会入队并出队一次。
- **空间复杂度**：`O(n)`，最坏情况下队列中可能保存大量节点。

## 关键点

1. 空树的节点个数为 `0`。
2. 递归法的核心是：左子树节点数 + 右子树节点数 + 当前节点。
3. 层序遍历的核心是：每访问一个节点，计数加 `1`。
4. 当前代码采用通用计数方法，没有利用完全二叉树性质做额外优化。
