# 559. N 叉树的最大深度

- **平台**：LeetCode
- **链接**：https://leetcode.cn/problems/maximum-depth-of-n-ary-tree/description/
- **难度**：Easy
- **标签**：树、深度优先搜索、广度优先搜索

## 题目描述

给定一个 N 叉树，找到其最大深度。

最大深度是指从根节点到最远叶子节点的最长路径上的节点总数。

N 叉树的每个节点可以拥有任意数量的子节点，这一点和二叉树只有 `left`、`right` 两个孩子不同。

## 示例

**示例 1：**

```text
输入：root = [1,null,3,2,4,null,5,6]
输出：3
```

**示例 2：**

```text
输入：root = []
输出：0
```

## 解题思路

本题和「104. 二叉树的最大深度」思路类似，只是从比较左右子树，变成遍历当前节点的所有子节点。

可以使用两种方式：

- 递归：求所有子树的最大深度，再加上当前节点这一层；
- 层序遍历：按层遍历整棵树，每遍历完一层，深度加 `1`。

### 方法一：递归法

#### 思路与算法

定义 `maxDepth(root)` 表示以 `root` 为根节点的 N 叉树最大深度。

递归终止条件：

```text
如果 root 为空，返回 0
```

对于非空节点，遍历它的所有子节点，递归求每个子树的最大深度，并记录最大值：

```text
depth = max(depth, maxDepth(child))
```

最后返回：

```text
depth + 1
```

其中 `+1` 表示当前节点自身这一层。

### 方法二：迭代法（层序遍历）

#### 思路与算法

使用队列进行广度优先搜索。

初始时，如果根节点不为空，就将根节点入队。然后按层处理队列：

1. 每轮循环开始时，记录当前队列长度 `size`；
2. 依次弹出当前层的 `size` 个节点；
3. 将每个节点的所有子节点加入队列；
4. 当前层处理完后，`depth` 加 `1`。

当队列为空时，说明整棵树已经遍历完成，`depth` 就是最大深度。

## 复杂度分析

设 N 叉树节点数为 `n`，树高为 `h`。

### 递归法

- **时间复杂度**：`O(n)`，每个节点都会被访问一次。
- **空间复杂度**：`O(h)`，主要来自递归调用栈。最坏情况下树退化成链状，空间复杂度为 `O(n)`。

### 迭代法（层序遍历）

- **时间复杂度**：`O(n)`，每个节点都会入队并出队一次。
- **空间复杂度**：`O(n)`，最坏情况下队列中可能保存大量同层节点。

## 关键点

1. 空树的最大深度为 `0`。
2. N 叉树没有固定的左右孩子，需要遍历 `children` 中的所有子节点。
3. 递归法的核心是：所有子树最大深度的最大值加 `1`。
4. 层序遍历的核心是：每处理完一层，深度加 `1`。
