# 104. 二叉树的最大深度

- **平台**：LeetCode
- **链接**：https://leetcode.cn/problems/maximum-depth-of-binary-tree/description/
- **难度**：Easy
- **标签**：树、深度优先搜索、广度优先搜索、二叉树

## 题目描述

给定一个二叉树 `root`，返回其最大深度。

二叉树的最大深度是指从根节点到最远叶子节点的最长路径上的节点数。

## 示例

**示例 1：**

```text
输入：root = [3,9,20,null,null,15,7]
输出：3
```

**示例 2：**

```text
输入：root = [1,null,2]
输出：2
```

**示例 3：**

```text
输入：root = []
输出：0
```

## 解题思路

本题要求树的最大深度，也就是从根节点向下走到最远叶子节点所经过的节点数量。

可以从两个角度理解：

- 使用递归时，当前节点的最大深度等于左右子树最大深度的较大值再加 `1`；
- 使用层序遍历时，每遍历完一层，深度就增加 `1`。

### 方法一：递归法

#### 思路与算法

定义函数 `getDepth(node)` 表示以 `node` 为根节点的树的最大深度。

递归终止条件：

```text
如果 node 为空，深度为 0
```

递归处理：

```text
leftDepth = getDepth(node.left)
rightDepth = getDepth(node.right)
depth = 1 + max(leftDepth, rightDepth)
```

也就是说，一个非空节点的最大深度由它的左右子树深度决定，取较大值后加上当前节点这一层。

### 方法二：迭代法（层序遍历）

#### 思路与算法

使用队列进行广度优先搜索。

初始时将根节点入队。每一轮循环处理队列中当前层的所有节点：

1. 当前层开始时，先记录队列长度 `size`；
2. 处理完这 `size` 个节点，说明当前层遍历完成；
3. 每完成一层，`depth` 加 `1`；
4. 遍历过程中，将下一层的非空左右孩子加入队列。

当队列为空时，所有层都已经遍历完成，`depth` 就是二叉树的最大深度。

## 复杂度分析

设二叉树节点数为 `n`，树高为 `h`。

### 递归法

- **时间复杂度**：`O(n)`，每个节点都会被访问一次。
- **空间复杂度**：`O(h)`，主要来自递归调用栈。平衡二叉树为 `O(log n)`，最坏情况下树退化成链表，为 `O(n)`。

### 迭代法（层序遍历）

- **时间复杂度**：`O(n)`，每个节点都会入队并出队一次。
- **空间复杂度**：`O(n)`，最坏情况下队列中可能保存一层中的大量节点。

## 关键点

1. 空树的最大深度为 `0`。
2. 递归法的核心是后序思想：先得到左右子树深度，再计算当前节点深度。
3. 层序遍历的核心是按层处理：每处理完一层，深度加 `1`。
4. 最大深度统计的是节点数，不是边数。
