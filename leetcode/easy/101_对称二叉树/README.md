# 101. 对称二叉树

- **平台**：LeetCode
- **链接**：https://leetcode.cn/problems/symmetric-tree/description/
- **难度**：Easy
- **标签**：树、深度优先搜索、广度优先搜索、二叉树

## 题目描述

给你一个二叉树的根节点 `root`，检查它是否轴对称。

所谓轴对称，是指二叉树的左子树和右子树互为镜像：

1. 两个根节点的值相同；
2. 左子树的左孩子要和右子树的右孩子相同；
3. 左子树的右孩子要和右子树的左孩子相同。

## 示例

**示例 1：**

```text
输入：root = [1,2,2,3,4,4,3]
输出：true
```

**示例 2：**

```text
输入：root = [1,2,2,null,3,null,3]
输出：false
```

## 解题思路

本题不能只比较左右节点是否相等，而是要比较两棵子树是否互为镜像。

核心比较关系是：

- `left.val == right.val`
- `left.left` 和 `right.right` 对称
- `left.right` 和 `right.left` 对称

### 方法一：递归法

#### 思路与算法

定义函数 `compare(left, right)`，判断两个节点为根的子树是否互为镜像。

递归终止条件：

1. `left` 为空、`right` 不为空：不对称，返回 `false`；
2. `left` 不为空、`right` 为空：不对称，返回 `false`；
3. `left` 和 `right` 都为空：对称，返回 `true`；
4. 两个节点都不为空，但值不同：不对称，返回 `false`。

如果两个节点都不为空且值相同，再继续比较外侧和内侧：

```text
外侧：left.left  和 right.right
内侧：left.right 和 right.left
```

只有外侧和内侧都对称，当前两棵子树才对称。

### 方法二：迭代法（使用队列）

#### 思路与算法

队列中每次放入一组需要比较的节点。初始时放入 `root.left` 和 `root.right`。

每次从队列中取出两个节点：

1. 如果两个节点都为空，说明这组节点对称，继续；
2. 如果只有一个为空，或者两个节点值不同，返回 `false`；
3. 否则继续按镜像顺序入队：
   - `leftNode.left` 和 `rightNode.right`
   - `leftNode.right` 和 `rightNode.left`

队列为空时，说明所有节点都满足镜像关系，返回 `true`。

### 方法三：迭代法（使用栈）

#### 思路与算法

栈的写法与队列类似，本质都是成对比较节点。

初始时压入 `root.left` 和 `root.right`，之后每次弹出两个节点进行比较。若节点有效且值相等，则继续按镜像关系压栈：

```text
leftNode.left   与 rightNode.right
leftNode.right  与 rightNode.left
```

只要比较过程中出现结构不一致或节点值不同，就返回 `false`。

## 复杂度分析

设二叉树节点数为 `n`，树高为 `h`。

### 递归法

- **时间复杂度**：`O(n)`，每个节点最多被访问一次。
- **空间复杂度**：`O(h)`，主要来自递归调用栈。平衡二叉树为 `O(log n)`，最坏情况下树退化成链表，为 `O(n)`。

### 迭代法（队列 / 栈）

- **时间复杂度**：`O(n)`，每个节点最多入队或入栈一次。
- **空间复杂度**：`O(n)`，最坏情况下队列或栈中可能保存较多节点。

## 关键点

1. 判断对称不是比较同侧节点，而是比较镜像位置的节点。
2. 递归比较顺序是：`left.left` 对 `right.right`，`left.right` 对 `right.left`。
3. 队列和栈版本的核心也是“成对取出、成对放入”。
4. 空节点也要参与比较，否则无法判断树的结构是否对称。
